# Health Vault Backend

Express.js API server for Health Vault application.

## Setup
```bash
npm install
cp .env.example .env
# Edit .env with your configuration
npm run dev
```

## Environment Variables

See `.env.example` for required variables.

## Database Seeding
```bash
node src/seed.js
```

Creates 8 test users with sample data.
